# Student Management System

This project is a simple Student Management System developed using Python.

## Features
- Add Student Record
- View Student Record
- Search Student
- Update Student
- Delete Student

## Technologies Used
- Python
- File Handling
- Basic OOP Concepts

## How It Works
This system stores student data in files and allows the user to perform operations like adding, viewing, updating, searching, and deleting student records.
